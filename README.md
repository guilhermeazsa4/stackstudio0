# Stack

One-page institucional da Stack — estúdio digital que cria sites e experiências jogáveis (jogos no Roblox, via Stack Interactive).

🔗 **Site:** [stackstudio0.vercel.app](https://stackstudio0.vercel.app/)

## Sobre

A Stack é um pequeno estúdio criativo. Sem camadas de gerência entre você e quem cria — direção de arte, desenvolvimento e produção de jogos, tudo próximo.

O site cobre:

- **Hero** — abertura da marca
- **Portfólio** — cases de sites entregues
- **Stack Interactive** — jogos publicados no Roblox (RUSH, Elemental Arena)
- **Equipe** — quem faz
- **Contato** — WhatsApp, e-mail e Instagram

## Stack técnica

- [Next.js 15](https://nextjs.org/) (App Router)
- React 19 + TypeScript
- Tailwind CSS + CSS customizado (`app/globals.css`)
- [Lenis](https://github.com/darkroomengineering/lenis) para smooth scroll

## Rodando localmente

```bash
npm install
npm run dev
```

Abre em [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
